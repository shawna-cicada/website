# Content Entry Guide

What to enter in the Studio (cicadaagility.com/admin) to make the site
fully yours, in the order that unblocks the most. Written for the
content owner, not a developer. For the article-writing workflow, see
docs/EDITOR_GUIDE.md.

## 1. People (once)

**Dashboard → People.** Add Shawna and Julia: name, role, photo
(replaces the placeholder portraits), short bio (replaces the drafts
marked "pending editorial review"), LinkedIn profile link.

## 2. Categories (once)

**Dashboard → Categories.** Suggested starters: Leadership, Founder
Growth, Organizational Effectiveness, Product & Delivery, AI Working
Norms, Case Examples. Categories power related-content suggestions.

## 3. Homepage content

**Dashboard → Update Homepage Content.** Choose the featured article and
featured assessment. Headline and supporting copy can also be updated
here — the tagline is part of the brand, change it deliberately.

## 4. Assessments

**Dashboard → Manage Assessments.** For each assessment: name, what it
tells people, who it is for, how long it takes, the button text, and —
when the provider link is ready — the assessment link. Leave the link
empty until then; the site shows "available soon" honestly. Mark exactly
one assessment as featured. The privacy note is shown before people
leave the site; keep it accurate for the provider you use.

## 5. Client logos

**Dashboard → Manage Client Logos.** Upload the logo, write a short
description (screen readers announce it), pick the company-size group.
**The logo will not appear anywhere until you check the written-
permission box and switch "Show on the website?" on.** This is
deliberate — never show a client without permission in writing.

## 6. Articles and videos

Three pieces reflecting the new positioning are the launch goal
(brief: "Initial Content Priorities"). Use **Create a New Article** with
the templates; the full workflow is in EDITOR_GUIDE.md. Migrated Wix
articles should keep their original web addresses — if an old address
must change, tell the site maintainer so a redirect is added.

## 7. What you never enter

- Page addresses, search titles, reading times — generated automatically.
- Booking links and assessment provider URLs — these live in the
  hosting configuration (ask the site maintainer), so they can't be
  broken by a typo in the CMS.
